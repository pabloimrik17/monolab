# Wealth React

React Router v7 wealth tracker. Displays live stock quotes using the Finnhub API.

## Environment Setup

```bash
cp apps/wealth-react/.env.example apps/wealth-react/.env
```

Edit `.env` and add your [Finnhub API key](https://finnhub.io/).

## Development

```bash
nx run @m0n0lab/wealth-react:dev
```
