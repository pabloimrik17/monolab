# Hello Experiments

This is the **experiments plugin** - a staging area for beta skills and commands in monolab.

## Purpose

Features in this plugin are experimental. They're being tested and refined before potentially graduating to production plugins.

## Current Experiments

### `/experiments:ralph`

Generate Ralph loop infrastructure for autonomous AI coding. Creates PRD, prompt template, and bash scripts from a project description.

```bash
/experiments:ralph "Build auth system with login and logout"
```

## Beta Disclaimer

Features here may:

-   Change without notice
-   Have incomplete documentation
-   Be removed if they don't work out

Stable features will graduate to their own dedicated plugins.
